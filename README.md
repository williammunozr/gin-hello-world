# Gin Web Framework

## Installation

```commandline
$ go get -u github.com/gin-gonic/bin
```

## Import it in your code

```commandline
import "github.com/gin-gonic/gin"
```

## Run the Gin hello world

```commandline
$ go run main.go
```
